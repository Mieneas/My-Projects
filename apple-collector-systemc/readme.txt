Hier wurde ein System in C++ mit Hilfe der Library SystemC entwickelt. Das System soll in einem Roboter eingesetzt werden.

Der Roboter soll Äpfel von unterschiedlichen Orten ernten können.
Die Äpfel befinden sich in verschiedene Höhen auf den Bäume.
Der Roboter verfügt über eine Batterie und einen Tank.
Alle Informationen (Bewegungsrichtung, Äpfel-Erkennung, Batteriestand, ...) empfängt das System durch geeignete Sensoren.
